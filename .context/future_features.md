## Backlog

### 1. 오늘의 약 체크용 위젯
- Android/iOS 홈 화면 위젯으로 앱 진입 없이 복약 체크
- 현재 시간대(아침/점심/저녁) 기반 자동 표시
- 위젯에서 바로 토글 가능 (원탭 체크)
- Flutter: `home_widget` 패키지 활용 예정

### 2. 약 등록 STT (Speech-to-Text) 기능
- 음성으로 약 이름, 갯수, 복용시기를 한 번에 등록
- 예: "고혈압약 30개 아침저녁" → 자동 파싱 후 등록
- Flutter: `speech_to_text` 패키지 활용 예정
- 파싱 로직: 숫자 + "개/정/포" + 시간대 키워드 매칭

### 3. 사진으로 약 등록 기능
- 약 이름 입력 없이, 약의 생김새(사진)로 등록
- 카메라 촬영 → 이미지 저장 → 약 카드에 썸네일 표시
- 이름 필드를 선택적(optional)으로 변경
- 추후 확장: 이미지 기반 약 식별 AI 연동 가능성
- Flutter: `image_picker` 패키지 (이미 OCR용으로 의존성 준비됨)

### 4. 복용주기 '기타' 세분화
- '기타' 클릭 시 3가지 카테고리 표시:
  - **2일마다 (격일)**
  - **특정 요일** (월~일 중 선택)
  - **필요할 때만** (비상약)
- 특정 요일 선택 시 → 요일 선택 UI 표시 (월/화/수/목/금/토/일 토글 칩)

#### Depth 3단계 문제 대안

> 복용주기 → 기타 → 특정요일 → 요일선택 으로 depth가 깊어지는 문제

**권장안: 인라인 펼침(Expandable Inline)**
- '기타' 클릭 시 바텀시트/새 화면 없이, 같은 폼 내에서 3가지 옵션이 **아래로 펼쳐짐**
- '특정 요일' 선택 시 바로 아래에 요일 토글 칩이 **애니메이션으로 나타남**
- 결과: 사용자가 한 화면 안에서 모든 설정 완료 (depth 1단계로 유지)

```
복용주기
[매일(기본)] [기타]
              ↓ 펼침
        ○ 2일마다 (격일)
        ● 특정 요일
           [월][화][수][목][금][토][일]  ← 바로 표시
        ○ 필요할 때만
```

- 데이터 모델: `Medication`에 `frequencyType` + `selectedDays: List<int>?` 추가
