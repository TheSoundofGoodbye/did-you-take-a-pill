

# Rule: Elderly-Friendly UI Design (고령자 친화 UI)
- **TARGET AUDIENCE**: 이 앱의 주요 사용자는 고령자(65세 이상)입니다. 모든 UI 컴포넌트는 이를 전제로 설계해야 합니다.
- **FONT SIZE**: 본문 텍스트 최소 16px, 안내/가이드 텍스트 최소 18px, 제목/헤딩 최소 22px. 정보 라벨 최소 14px.
- **TOUCH TARGET**: 터치 가능한 요소(버튼, 체크박스, 토글)의 최소 크기는 **56×56px** 이상.
- **CONTRAST**: 모든 텍스트와 배경의 대비(contrast ratio)는 최소 4.5:1 이상 유지.
- **BUTTON**: 주요 액션 버튼은 충분히 크고(높이 56px 이상), 텍스트는 18px 이상, 눈에 띄는 색상 사용.
- **SPACING**: 요소 간 간격(margin/padding)을 넉넉히 배치하여 오터치를 방지.
- **POSITION**: 주요 버튼은 화면 **중하단(center-bottom)** 에 배치하여 손이 닿기 쉽게.
- **NO ABSTRACT ICONS**: 톱니바퀴(⚙), 햄버거 메뉴(☰), 점 3개(⋮) 등 IT에 익숙하지 않은 사용자가 이해하기 어려운 추상적 아이콘을 단독으로 사용하지 말 것. 반드시 **텍스트 라벨과 함께** 표시하거나, 텍스트 버튼으로 대체할 것.
- **ICON GUIDELINE**: 아이콘 사용 시 반드시 텍스트와 병행. 예: `[+ 약 추가]` 버튼은 허용, 단독 `[+]` 아이콘은 금지.
